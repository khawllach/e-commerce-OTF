# 🛍️ OTF E-Commerce

A modern and responsive e-commerce web application built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**. The project focuses on delivering a clean shopping experience with a modern UI, responsive layouts, and reusable components.

> This project was developed to strengthen my frontend development skills and demonstrate my ability to build modern, scalable user interfaces.

---




## ✨ Features

- 🛒 Browse products
- 🔍 Product search
- 🏷️ Product categories
- ❤️ Wishlist
- 🛍️ Shopping cart
- 📱 Fully responsive design
- ⚡ Fast loading with Vite
- 🎨 Modern UI/UX
- ♻️ Reusable React components

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/khawllach/e-commerce-OTF.git
```

### Navigate to the project

```bash
cd e-commerce-OTF
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
